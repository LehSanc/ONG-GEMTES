# ONG-GEMTES
  Este projeto busca cumprir com a proposta da ONG GEMTES (Grupo de Educação Multidisciplinar em Tecnologias Socioambientais), que trabalha com práticas educacionais. De forma resumida, a principal finalidade desse software é o gerenciamento de informações que serão inseridas em um banco de dados, bem como a concepção de relatórios a partir desse conteúdo. 
  Nessa instituição são realizadas consultas periódicas entre crianças (assistido) que estão na primeira infância e em fase escolar, e psicopedagogos. No primeiro momento, será efetuado um questionário a fim de colher dados relevantes para o diagnóstico, planejamento e encaminhamento ao tratamento necessário, além da rotina conduzida pelo profissional.
  O sistema proposto visa auxiliar essa rotina de consultas e tratamentos de modo que o psicopedagogo (usuário) consiga: gerar o questionário a ser aplicado de acordo com as condições do caso (idade do assistido, área de aplicação); inserir os dados gerados com esse questionário e armazenar; produzir relatórios baseados nas respostas gravadas; e documentar comentários e avaliações sobre cada consulta efetuada.
